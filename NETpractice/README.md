## DESC

NetPractise is a 42 curriculum project that focuses on the fundamentals of networking.

the goal with this project is to learn the basics of concepts such as TCP/IP addressing through practical exercises.

in each level of the simulation, the objective is restore communication between devices by configuring :

- IP addresses
- subnet masks
- default gateways
- router interfaces
